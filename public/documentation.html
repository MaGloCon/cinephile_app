Cinephile API Documentation

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic Endpoint URL HTTP Method Format of Request Data Format of Response Data

Add movies to user's favorite list /users/:id/:title POST None A message returning if a movie has been successfully added to the list.

Delete a user /users/:id/ DELETE None A message returning if the user was successfully deleted.
