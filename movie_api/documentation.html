Cinephile API

A REST API that provides access to a database of movies, directors, and actors.

Features

  • Return a list of ALL movies to the user
  • Return data: description, genre, director, image URL, whether it's featured or not
  • Optional: actors, release year
  • Return data about a genre (description) by name/title
  • Return data about a director (bio, birth year, death year) by name
  • Return data about a director (bio, birth year, death year) by name
  • Allows users to register (username, password, email, date of birth)
  • Allows users to update their user info (username, password, email, date of birth)
  • Allows users to add a movie to their list of favorites
  • Optional: allows users to add movie to their watchlist
  • Allows users to remove a movie from their list of favorites
  • Optional: allows users to remove a movie from their watchlist
  • Allows existing users to deregister

Technical Requirements

  1. created with Node.js and Express.js
  2. uses REST architecture, with URL endpoints corresponding to data operations listed above
  3. uses at least 3 different middleware modules, such as the body-parser package for reading data from requests and morgan for logging.
  4. uses "package.json"
  5. database built using MongoDB
  6. provides movie information in JSON format
  7. tested in Postman
  8. includes user authentication and authorization code
  9. includes data validation logic
  10. source code is deployed to GitHub
  11. deployed to Heroku
